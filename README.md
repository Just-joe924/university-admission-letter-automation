# university-admission-letter-automation
A full-stack system for verified admission workflows and PDF document generation designed to automate the generation and distribution of university admission letters.

# The application enables:
-Verified student data intake
-Dynamic PDF generation using templated admission letters
-Automated email delivery with attachments
-Admin-level analytics and monitoring dashboard

#Tech stack
Frontend: React + Tailwind CSS
Backend: Node.js + Express
Database: MongoDB or PostgreSQL
PDF generation: Puppeteer
Email: Nodemailer
Authentication: JWT
